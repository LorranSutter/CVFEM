RÁPIDAS INSTRUÇÕES PARA UTILIZAÇÃO DOS CÓDIGOS:

->> Códigos fonte:
 -> geometry.py
    - código contendo algumas funções geométricas necessárias para a estruturação dos pontos da malha.
 -> leGmsh.py
    - código encarregado de ler arquivos .msh gerados pelo software Gmsh e gerar arquivos .dat
    - tipo de geometria aceita:
      - somente estruturas bidimensionais
      - podem ser formadas por retas e/ou quaisquer tipo de curva
      - só podem possuir um contorno, ou seja, geometrias com buracos não são aceitas
 -> cvfem.py
    - código encarregado de ler arquivos .dat gerados pelo código acima
    - contém funções necessárias para a resolução de porblemas bidimensionais de advecção e difusão em geometrias complexas

->> Utilização:
 -> rodar o código leGmsh.py com entrada de um arquivo .msh gerado pelo software Gmsh
 -> rodar o código cvfem.py com entrada do arquivo .dat gerado pelo código leGmsh.py

OBS: os arquivos teste.msh e saidaTeste.dat foram utilizados no trabalho de conclusão de curso.
